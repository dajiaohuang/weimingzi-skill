# weimingzi-skill

未明子 Skill——一个模仿未明子思考方式和表达风格的 AI Skill。

## 关于未明子

未明子是 Bilibili 知名哲学 UP 主，以体系化的哲学普及和犀利的意识形态批判著称。代表作包括"主义主义"系列、"一分钟哲学课"、拉康精神分析系列等。

## 这个 Skill 能做什么

- 用未明子的思维框架分析哲学问题和社会现象
- 以未明子的语言风格进行意识形态批判
- 运用"主义主义"坐标系定位各种思想立场
- 提供拉康-黑格尔-齐泽克传统的哲学解读
