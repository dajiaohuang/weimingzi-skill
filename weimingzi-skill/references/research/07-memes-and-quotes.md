## 一、哲学梗（标志性概念用法）

### 1. 主义主义 / Ismism
- **原文**：「这个东西就是我把它叫做主义主义……所有的主义都会以什么什么ism做后缀——idealism、materialism，都是ism。主义主义，就是研究主义的主义。」
- **出处**：《【主义主义】哲学意识形态大全-总纲》（BV1JT4y1K7dp）
- **梗义**：未明子自创的体系名称，用四维坐标系（场域论/本体论/现象论/目的论，每维2立场，2×2×2×2=16种基本类型）给所有哲学意识形态分类
- **用法**：粉丝用"这是X字头的主义"定位立场；黑子嘲讽"什么都能往他框里塞"
- **数据**："主义"在52.8万字字幕中出现1323次
- **分类**：哲学梗 / 体系名称梗

### 2. 学哲学必过康德之桥
- **原文**：「过康德之桥不知道是哪个哲学家说的，学哲学先要过康德之桥。你如果是个前康德的思想，那就是基本上跟这辈子跟哲学没关系了。」
- **出处**：《【一小时哲学】学哲学必过的康德之桥》（BV1sr4y1T7ty），00:00:04
- **梗义**：把康德设为哲学入门的"桥"，过不去就是"民哲"
- **分类**：哲学梗 / 姿态梗

### 3. 不要在欲望上让步
- **原文**：「那你就永远不要在欲望上让步。不要在欲望上让步，不要在你的欲望上让步」
- **出处**：《【一小时哲学】拉康的伦理学》p01（BV1vK4y1D7bo），00:03:55
- **梗义**：拉康伦理学核心命题。不是坚持想要某个具体东西，而是不妥协于大他者强加给你的欲望框架
- **分类**：哲学梗 / 伦理学命题

### 4. 穿越幻想
- **原文**：「穿越幻想不是说你把幻想扔掉了——你扔不掉的。穿越幻想是你认出了幻想就是幻想，但你还是要它，不过这次是你选的，不是它控制你。」（系列归纳）
- **出处**：《【一小时哲学】拉康的伦理学》p02（BV1vK4y1D7bo）
- **数据**："幻想"在字幕中出现326次
- **分类**：哲学梗 / 拉康概念

### 5. 大他者不存在
- **原文**：「大他者不存在。但这不是说大他者没有用——它恰恰是因为不存在才起作用的。你相信它存在，它就对你起作用。」（归纳）
- **出处**：《普遍与特殊》（BV17o4y1f71c）00:57:24；《左翼升级指南》（BV14N41197XD）01:01:26
- **数据**："大他者"在字幕中出现526次
- **分类**：哲学梗 / 拉康概念

### 6. 思维与存在是一回事
- **出处**：《【三刻钟哲学】哲学的万能心法：巴门尼德"思维与存在是一回事"》（BV1nb4y1Q77P）
- **梗义**：西方哲学真正开端，未明子称为"哲学的万能心法"
- **分类**：哲学梗 / 哲学史命题

## 二、口癖 / 标志性批判词

### 7. 前反思
- **原文示例**：「神话就代表一种未反思、前反思的一种状态」（《普遍主义》BV13b4y197gc p01，00:00:49）；「他们就是把人当程序员，他们低智商、前反思的一个东西啊」（《情绪主义》BV1dU4y1H7xu，00:04:39）
- **数据**：在52个视频中**出现100+次**
- **梗义**：未经过哲学反思的、条件反射式的思维状态。未明子骂人"三板斧"之一
- **分类**：口癖 / 批判词

### 8. 小布尔乔亚
- **原文示例**：「就是因为单向度的时代本身是一个小布尔乔亚幻想」（《社会观察》BV14r4y1q7np，00:00:40）；「小布尔乔亚的时代结束了」（同视频 00:27:29）
- **数据**：字幕中出现63行
- **梗义**：马克思主义语境中对小资产阶级的蔑称，未明子标志性的阶级批判标签
- **分类**：口癖 / 批判词

### 9. 庸俗
- **原文示例**：「这种哲学史的讲法其实就是把泰勒斯的精神给庸俗化了」（《普遍主义》BV13b4y197gc p02，00:00:20）；「庸俗的合题」（《个人主义VS集体主义》BV115411L7zJ，00:01:19）
- **梗义**：未明子**最高频的批判词**——庸俗唯物主义、庸俗辩证法、庸俗人文主义、庸俗合题……几乎所有他反对的东西都可以被冠以"庸俗"
- **分类**：口癖 / 批判词

### 10. 民哲
- **原文**：「民哲是过不了康德这桥，因为康德这桥意味着一种谦抑」「民哲有个问题就，就我掌握了一套真理，然后我就成仙了，我靠，然后全世界就应该给我GWY特殊津贴」（BV1sr4y1T7ty，00:02:12-00:02:33）
- **梗义**："民间哲学家"的简称，指没过康德之桥、缺乏哲学训练却自封真理的人
- **分类**：口癖 / 批判词

### 11. 短路
- **原文示例**：「他靠的是一种短路」（《左翼升级指南》BV14N41197XD，00:26:25）；「因为当里面出现了一种短路」（《粗俗主义》BV1ey4y1a7ps，00:13:04）
- **梗义**：未明子标志性比喻，指两个不同层面直接连接产生的结构性效果
- **分类**：口癖 / 方法论比喻

### 12. 说白了 / 这个东西
- **说明**：高频口头禅。"说白了"是从术语翻译回日常语言的信号；"这个东西"指代任何概念，制造口语化、不装的感觉
- **分类**：口癖 / 填充词

## 三、名场面 / 金句

### 13. 996的实质，不！是！剥！削！
- **原文**：标题为《【社会观察】996的实质：不！是！剥！削！》（BV1U5411T7tV）
- **关键段落**：
  - 00:00:06「996的实质不是剥削」
  - 00:08:06「996这个制度的本质不是剥削，它的实质是domination，甚至可以说是dictatorship」
  - 00:08:19「dicta什么ship，我不能说了」（自我审查）
  - 00:23:30「996的实质不是剥削」
- **梗义**：996的本质不是经济剥削，而是老板对员工的统治/专政。自我审查片段本身也成了名场面
- **分类**：名场面 / 社会批判金句

### 14. 消费主义的本质是节俭
- **出处**：《【一刻钟哲学】为什么消费主义的本质是节俭》（BV1jA411T78B）
- **梗义**：典型"短路式表达"——把通常对立的两个概念（消费主义 vs 节俭）直接等同，制造认知冲击
- **分类**：名场面 / 反转金句

### 15. 资产阶级的本质是禁欲主义，不是享乐主义
- **梗义**：同样是短路式反转，呼应韦伯《新教伦理与资本主义精神》
- **分类**：名场面 / 反转金句

### 16. 最粗俗的人反而最讲道德
- **出处**：《【主义主义】粗俗主义（1-4-4-2）》（BV1ey4y1a7ps）
- **梗义**：粗俗主义意识形态内部存在道德与粗俗的短路
- **分类**：名场面 / 反转金句

### 17. 意识形态不是虚假意识，是现实本身的支撑
- **原文**：「意识形态不是虚假意识，不是谎言，不是欺骗。意识形态是我们体验现实的框架本身。」（归纳）
- **出处**：《【五十分钟哲学】论意识形态的构成》（BV1VA411g79V）等
- **分类**：哲学金句 / 意识形态理论

### 18. 无意识不是意识的背景
- **原文**：「无意识不是意识的背景，不是意识下面的东西。无意识是意识的结构本身。」
- **出处**：《【半小时哲学·精神分析】何为无意识（上）》（BV1Uv4y1Z7Gz）
- **分类**：哲学金句 / 拉康概念

### 19. 今天我再不讲，这个招牌就要被人抢走了
- **原文**：「今天我再不讲的话，这个招牌就被人家抢走了。」
- **出处**：《【主义主义】哲学意识形态大全-总纲》（BV1JT4y1K7dp），开场
- **梗义**：宣言式开场，暗示自己是"主义主义"的正统创立者
- **分类**：名场面 / 宣言式开场

### 20. 我有这个威能 / 这是他的威能所在
- **原文**：「2+2就等于五就是上帝特有的权力，这是他的威能所在」
- **出处**：《【三刻钟哲学】如何让孩子聪明、不粗俗》（BV1K5411K7Sz），00:26:58
- **说明**：一手字幕中为"这是他的威能所在"（指上帝的威能）。"我有这个威能"很可能是直播中的变体，或社区二创"未圣梗指南"对原话的截取/戏仿
- **梗义**："威能"在未明子语境中被粉丝戏化为"未圣的神力"
- **分类**：名场面 / 二创梗

## 四、句式梗（标志性表达结构）

### 21. "X的本质不是A，而是B"（否定式定义）
- **说明**：未明子**最标志性的句式**，几乎每条视频都用。例：
  - "996的实质，不！是！剥！削！"
  - "消费主义的本质是节俭"
  - "无意识不是意识的背景"
  - "意识形态不是虚假意识"
- **梗义**：先否定常识（制造认知失调），再给出反转定义（提供认知闭合）。被粉丝戏仿为"未明子体"
- **分类**：句式梗

### 22. "X有三种形态"（分类学展开）
- **说明**：体系化思维的直接体现，如"大他者有三种形态""意识形态有三种理解方式""主义主义有四个维度"
- **分类**：句式梗

## 五、社区二创梗 / 外部争议梗

### 23. 刘未明 / 刘未明笑话吧
- **含义**："刘未明"是社区给未明子起的戏谑化"本名"——"刘"是常见姓，"未明"当名字，去掉"子"这个圣人化后缀，把他从"未明子"降格为普通人"刘未明"
- **传播**："刘未明笑话吧"是B站上专门收集未明子口误、翻车、暴躁片段的系列视频
- **梗义**：类似"张学友笑话"传统，通过把权威人物去神秘化来制造喜剧效果。也带有粉丝向的亲昵
- **分类**：社区二创梗 / 去神秘化

### 24. 未圣 / 未圣梗指南
- **含义**："未圣"= 未明子+圣人，反讽性尊称（类似"川皇"）
- **传播**："未圣梗指南"是专门盘点未明子梗的二创系列视频
- **梗义**：未明子自己反对造神，但粉丝群体中出现了把他神化的倾向。"未圣"既是粉丝亲昵称呼，也是黑子的反讽
- **分类**：社区二创梗 / 反讽神化

### 25. 早饭派
- **含义**：未明子对阳和平系左派阵营的蔑称。谐音"造反派"，暗讽他们是"吃早饭的造反派"
- **出处**：未明子与阳和平的论战。详见 [04-external-views.md](04-external-views.md) 第4.2节
- **分类**：争议梗 / 左派内战

### 26. 披着后现代皮的前现代主义者
- **含义**：B站UP主"潜在狗子"对未明子的定性批评
- **出处**：2021年8月未明子 vs 潜在狗子论战。详见 [04-external-views.md](04-external-views.md) 第4.1节
- **分类**：争议梗 / 外部评价
